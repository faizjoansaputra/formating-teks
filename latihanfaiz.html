Apa itu html

HTML adalah sebuah bahasa yang menggunakan markup atau penanda+ untuk membuat halaman web.Penanda atau markup ini, nanti akan kita sebut dengan HTML berperan untuk menentukan struktur dan tampilan dari sebuah web kalau kita ibaratkan nih HTML itu seperti batu untuk membangun rumah. Batu bata ini dapat disusun,hingga menjadi fondasi dasar.HTML memiliki kelebihan sepereti memiliki sumber memiliki sumber komunitas yang cukup besar dengan penggunaan yang luas. tidak hanya itu,kelebihan lainya HTML mudah untuk dipelajari oleh seorang web developer pemula kelebihan lainnya adalah HTML merupakan bahasa pemograman yang gratis atau open source
